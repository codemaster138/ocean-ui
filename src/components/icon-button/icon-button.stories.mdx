# Icon Button

An Icon Button is a CTA just like a button, except that it _only_ contains an icon. In terms of choosing whether to use the primary or secondary style, the same rules apply as for the regular button.

If the icon button in placed on the default background color, add the `light` prop. Remove it if placed on `accent_4`.

## Badges
A badge is a small element containing a small integer or *very short* text. It should be used, for example, when displaying the number of notifications a user has received since last checking.

Set the `badge` option to any string or number to display it in the badge. If you wish to customize the color, use the `badgeColor` option. The `badgeColor` option may be any or key from the theme object, or a hex value, the latter of which is **not recommended** and should never be used.

The default badge color is the primary color. It is recommended to use `accent_1` as `badgeColor` when the IconButton is primary.
