# Input

The input component can be used to receive user input in the form of text, numbers or passwords. *An input should always have a label (Set via the `label` prop)!*

The input provides multiple props that might be of interest to developers. Most importantly, `update` and `value`. This following code example shows how one can use the `Input` component like a normal input:

```tsx
import { Button, Input, Card, CardForm, h1 } from "@ocean-ui/ocean";
import { useState } from "react";

export default function MyForm() {
  const [name, setName] = useState("");

  return (
    <CardForm
      noValidate
      onSubmit={() => {
        /* Submit form */
      }}
    >
      <Card
        footer={
          <>
            <Button>Exit</Button>
            <Button
              type="submit"
              primary
              onClick={() => {
                /* Submit form */
              }}
            >
              Submit
            </Button>
          </>
        }
      >
        <h1>Simple form</h1>
        <Input label="Your name" type="text" value={name} update={setName} />
      </Card>
    </CardForm>
  );
}
```

Each time the content changes, `update` is called, with the updated content.
